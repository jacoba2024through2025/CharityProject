CodeFund's One Month Coding Challenge Charity

We raised two million USD for school computer science classes in 2023. Codefund starts its 2024 charity September 1, 2024 and ends September 30, 2024.

CodeFund's Purpose With This Charity

This charity aims to give students an interest in learning more about computer science. Codefund is asking anyone to complete coding challenges in the following languages:

How does the charity work?

One programming language project completed counts as a donation. For example, if someone completes the Python project, one donation will be added. 100% of the proceeds from this charity go to underdeveloped schools that may not have the ability to fund computer science courses.

Project Requirements Donation Amounts For Each Project
For the Python project, you will research ways you can have a positive impact on your community and incorporate it into a Python project. If you create one project, your donation will be 30 USD.
For the Java project, you will create a lesson plan structure for a computer science school. If you complete all three projects, your donation is worth 100 USD.
For the Ruby project, you will create an interactive website that allows students to find different career paths. Email us at codefund@gmail.com to submit your project for review.
